# Multi-Carrier Shipping Rate Comparator

An autonomous web agent that compares shipping rates across multiple carriers (FedEx, UPS) in real-time.

## 🚀 What It Does

1. Takes package dimensions and destination from user
2. Simultaneously navigates to FedEx and UPS rate calculators
3. Fills out complex multi-step forms
4. Extracts and compares shipping rates
5. Returns cheapest and fastest options

## 🛠️ Technology Stack

- **TinyFish Web Agent API** - Core browser automation
- **Python 3.8+** - Application logic
- **Asyncio** - Parallel carrier queries

## 📋 Setup Instructions

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Get TinyFish API key from https://agent.tinyfish.ai/api-keys
4. Create `.env` file with your API key
5. Run: `python main.py`
